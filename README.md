## 2. Задатак

Програм који на основу унете дужине **полупречника** круга *r* израчунава:

- **Обим** круга O  
- **Површину** круга P  

### Формуле:
- Обим:  
  **O = 2πr**

- Површина:  
  **P = πr²**
